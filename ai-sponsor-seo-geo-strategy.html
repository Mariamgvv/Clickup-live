The reference tool — "SEO + GEO Autopilot"

A dashboard that runs a continuous loop — Audit → Diagnose → Strategize → Fix → Re-audit — across both Google Search and AI engines. Four modules:

1 · The Auditor
Technical crawl + live Google ranking (Search Console) + Core Web Vitals + an AI-search readiness score (0–100); surfaces top 5 problems.
2 · The AI-Search Spy
Tests whether AI engines actually cite your site; scores AI-search visibility across 5 platforms; flags reasons for non-inclusion.
3 · The Diagnoser
Finds striking-distance keywords (positions 8–20), maps top questions asked in AI search by intent, highlights content gaps.
4 · The Fixer
Generates deployment-ready code: FAQ schema (JSON-LD), llms.txt, robots.txt, rewritten titles, meta, answer blocks.
Is it doable for us? (Claude's assessment)

🟢 Doable now, essentially free (Claude + what we have)

  • The Fixer — JSON-LD FAQ schema, llms.txt, rewritten titles/meta/answer blocks. This IS the on-site AEO foundation.
  • Dashboard shell — static HTML + charts on GitHub Pages (our standard).
  • Basic audit — Core Web Vitals via Google's free PageSpeed API; Search Console (getaisponsor.com verified).

🟡 Doable but needs setup + small cost (Matt's call)

  • Keyword & search-volume data → DataForSEO (cheap, <~$1/1,000 lookups) or Semrush/Ahrefs.
  • Search Console at scale → API + OAuth setup.
  • AI-Search Spy (do ChatGPT/Perplexity/Gemini cite us?) → hardest piece; needs querying engines via APIs or a paid AI-visibility tool.
