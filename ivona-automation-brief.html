Ivona's Input — Her Words

So every week I create Summary Decks for Audos cohorts, which is honestly pretty time-consuming and repetitive. Right now, my process is:

  • taking the meeting recording,
  • using ChatGPT to create a summary following the same format every time,
  • using Gamma to turn it into a deck,
  • and then manually sending messages in each cohort with both the deck and the meeting recording.

Another big task is scheduling interviews for Founder in Residence candidates. I manually send messages to 70+ people on ZipRecruiter, suggesting time slots, then manually create a spreadsheet with their names and ZipRecruiter profiles, and after that I create separate Google Calendar events for each of them. There's a lot of manual work involved there as well.

These are some of the main repetitive tasks I'm currently handling that take up a significant amount of my time. If I think of anything else, I'll share it with you too.

Automation Breakdown
1. Summary Decks for Audos Cohorts
10 cohorts × weekly = ~10 decks per week
Almost Fully Automated
Current Process (2+ hrs/week)
  • Wait for meeting recording
  • Paste into ChatGPT, prompt for summary
  • Copy into Gamma, format as deck
  • Open each cohort channel manually
  • Send deck + recording link to each
  • Repeat for every cohort that week
With Automation (10 min/week)
  • Fathom posts summary to Slack automatically
  • Claude detects new cohort call
  • Claude generates formatted deck instantly
  • Claude posts to the correct cohort channel
  • Ivona reviews and approves
  • Done — for all 10 cohorts
  • ✅The Fathom bot already posts the full AI meeting summary to #meetings-fathom after every call — the hard part is already done
  • ✅Claude watches #meetings-fathom, detects cohort calls, and generates a formatted summary deck automatically — same quality as Gamma, no manual steps
  • ✅Claude posts the deck directly to the cohort's Slack channel with the recording link
  • ✅ChatGPT and Gamma are both replaced — no external tools needed
One thing to confirm: Are the cohort Slack channels in our internal Guru workspace, or in a separate Audos community workspace? If Guru workspace — Claude posts automatically. If Audos workspace — Claude still builds the deck, Ivona just clicks send.
